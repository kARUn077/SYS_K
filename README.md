# Kirtan's Portfolio

A modern, interactive portfolio website inspired by classic games like Project IGI. Showcases achievements, artworks, gaming stats, and more—with hidden easter eggs for retro gaming fans!

## Features
- Custom crosshair and night vision mode
- IGI-style easter egg (image + audio overlay, triggered by Up, Up, Down, Down)
- Achievements, artworks, and gaming stats
- Responsive design and smooth UI

## Easter Egg
Trigger the IGI easter egg by pressing:

```
Up, Up, Down, Down (arrow keys)
```

This will display a classic IGI guard image and play the iconic "Hey you, stop there!" audio.

## Setup & Run
1. Clone the repository:
   ```
   git clone <your-repo-url>
   ```
2. Place all assets (images, audio) in the `assets/` folder.
3. Open `index.html` or any page in your browser, or use a static server (like Vercel, Netlify, or Live Server extension).

## Deployment
- Works on any static hosting (Vercel, Netlify, GitHub Pages, etc.)
- All assets are relative to the `assets/` folder
